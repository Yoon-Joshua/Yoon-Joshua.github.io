笔记：Texture mapping

发表于2023-07-17|更新于2023-08-01|Vulkan

Images

纹理的创建和使用通常由以下几个步骤。

  • Create an image object backed by device memory
  • Fill it with pixels from an image file
  • Create an image sampler
  • Add a combined image sampler descriptor to sample colors from the texture

在学习swap chain相关的知识时，我们接触过image。不过，那些image是swap chain extension自动创建的。而在本章节中，我们将自己来创建image。

在给image对象填充数据的时候，我们一般有两种方式。一是先创建并填充staging image，然后将staging image的数据拷贝到image。二是先创建并填充staging buffer，然后将staging buffer的数据拷贝到image。

Preparing the texture image

Image刚被创建时，由于时空白的，我们不关心其中的内容，其布局为VK_IMAGE_LAYOUT_UNDEFINED。

为了将数据传输进image中，将其布局转换为VK_IMAGE_LAYOUT_TRANSFER_DST_OPTIMAL，即将image作为传输目标。

传输完毕后，image的数据要供shader读取，所以将其布局转换为VK_IMAGE_LAYOUT_SHADER_READ_ONLY_OPTIMAL。

Sampler

假如屏幕上有一个尺寸为100×100100\times100100×100的矩形，我们要把一张512×512512\times 512512×512图片贴在这个矩形上，此时矩形上的每一个像素应该是什么颜色呢？或者，这个图片的尺寸是64×6464\times 6464×64而不是512×512512\times 512512×512，此时矩形上的每一个像素应该是什么颜色呢？这就涉及到采样的问题。

  1. 所谓oversampling（过采样），就是纹理有着过少（比它要贴上去的图形的像素少）的纹素，贴图的分辨率相对较低的情况。针对oversampling，要么直接选取离fragment纹理坐标最近的纹素的颜色作为此fragment的颜色；要么使用滤波器，比如bilinear filtering（双线性过滤）。

  1. 所谓undersampling（欠采样），就是纹理有着过多（比它要贴上去的图形的像素多)的纹素，贴图的分辨率相对较高的情况。Undersampling会导致artifact的出现，其解决方案是anisotropic filtering（各向异性滤波器）。Anisotropic filtering的作用如下图所示。

  1. 除了oversampling和undersampling，还有addressing mode的问题。当fragment的纹理坐标超过了纹理的尺寸，fragment的颜色该如何确定呢？常见的addressing mode如下。

Anisotropy device feature

各向异性滤波是可选的设备特性。如要使用此特性，需要在创建逻辑设备的时候指出。另外，在创建逻辑设备之前，为保险起见，要检查设备是否支持anisotropy，尽管现代显卡基本上都支持这个特性。

Combined image sampler
